Разработка сервиса: abit-anketa
1. Зарегистрироваться на сайте github.org
После регистрации добавим в репозиторий abit-anketa
2. Установить git
3. Установить TortoiseGit
4. Установить Python 3.4
5. Установить Django 1.7.11
В командной строке pip install gjango==1.7.11
6. Установить MySQL WorkBench 5.2
Дополнительно в командной строке pip install mysql-connector
7. Установить Sybase PowerDesigner 15
8. Установить Sublime.Text2 или Pycharm
9. Скачать репозиторий abit-anketa 
В командной строке git clone https://github.com/voffan/abit-anketa.git
Или с сайта github.org кнопка "Clone or download"
